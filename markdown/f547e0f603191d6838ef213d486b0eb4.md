Getting Started
Every DeepAI API is a single `POST` request. Send the model's inputs as `multipart/form-data` and authenticate with your API key in a request header.
1. Get your API key
Sign in and copy your key from your account dashboard. API access requires a DeepAI Pro subscription. Keep your key secret — treat it like a password, and never expose it in client-side code.
2. Make a request
Pass your key in the `api-key` header and the model's inputs as form fields. The endpoint URL is `https://api.deepai.org/api/<model>`, where `<model>` is the model id (shown with each model below).

```
curl -X POST https://api.deepai.org/api/text2img \
    -H 'api-key:YOUR_API_KEY' \
    -F 'text=a serene mountain lake at sunrise'
```

3. Handle the response
On success you receive `200 OK` with a JSON body. `output_url` links to the generated file and `id` is the unique job identifier:

```
{
    "id": "59a0e8a9-...",
    "output_url": "https://api.deepai.org/job-view-file/.../output.jpg"
}
```

Errors
Errors return a non-200 status and a JSON body with an `err` or `status` message:
  * 401 — missing or invalid API key, or a usage quota was exceeded.
  * 410 — the requested model has been removed.
  * 500 — the model id was not found, or a processing error occurred.

## AI Image Generator
This is an AI Image Generator. It creates an image from scratch from a text description.
Endpoint: `POST https://api.deepai.org/api/text2img`
Parameters  
| Parameter  | Type  | Required  | Default  |  
| --- | --- | --- | --- |  
| text  | Yes  | —  |  
`text` — a string describing what to generate, e.g. "A starry night over a serene lake" or "A portrait of an old pirate".
`width`, `height` — pass a string, e.g. "256" or "768" (default 512). Use values between 128 and 1536, in multiples of 32. Recommended for standard images: 1024x576, 1024x720, 512x512, 768x1024, 576x1024. Recommended for HD images: 1216x832, 1152x896, 1024x1024, 896x1152, 832x1216. Values above ~700 or below 256 may produce strange outputs.
`image_generator_version` — "standard" (default), "hd", "genius", or "super_genius".
`resolution` — "2k" (default) or "4k". Only used when `image_generator_version` is "super_genius".
`genius_preference` — "anime", "photography", "graphic", or "cinematic". Only used when `image_generator_version` is "genius".
`negative_prompt` — a string describing what to remove from the image; useful for improving quality and detail. Example: bad anatomy, blurry, cropped, deformed, disfigured, duplicate, extra limbs, fused fingers, jpeg artifacts, low quality, lowres, mutated hands, out of frame, signature, text, watermark, worst quality.

## Waifu2x
Waifu2x is an algorithm that upscales images while reducing noise within the image. It gets its name from the anime-style art known as 'waifu' that it was largely trained on. Even though waifus made up most of the training data, this waifu2x api still performs well on photographs and other types of imagery. You can use Waifu2x to double the size of your images while reducing noise.
Endpoint: `POST https://api.deepai.org/api/waifu2x`
